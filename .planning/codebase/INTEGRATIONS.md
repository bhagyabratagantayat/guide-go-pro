# Codebase Integrations

## External APIs & Services
- **Cloudinary:** Image/document upload and storage (`cloudinary`, `multer-storage-cloudinary`)

## Databases
- **MongoDB:** Connected via `mongoose`

## Communication
- **Email:** `nodemailer` for transactional emails
- **Websockets:** `socket.io` for real-time bi-directional events (e.g., chat, tracking)

## Auth Providers
- Custom JWT-based authentication
